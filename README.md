# Project Title

OpenFrameworks FFT PostCard

## Getting Started

You'll need OpenFrameworks library.

### Sterring

1 Starts the music
2 Stops the music
3 Gives you camera coordinates and quaternion
4 Turns on/off the snow
5 Turns off/on collisions (if laggy)
